# Node JS Chat App


### DEMO
![Image](http://i.imgur.com/5qYl0gC.png)

## Installation
- `npm install `
- `node server.js`
- Open (http://localhost:3000)

## Features
- Login
- Create room
- Send private message
- Support for Website and Mobile
- Multiple Chat
- Show "is typing" when someone typing
